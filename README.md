How to setup

Prerequisites

Linux
Python 2.0 
NVIDIA GPU + CUDA 8.0 + CuDNNv5.1
TensorFlow 1.0


Getting Started

Clone this repo:
git clone https://github.com/duxingren14/DualGAN.git
cd DualGAN

Train the model
python main.py --phase train

Test the model:
python main.py --phase test
